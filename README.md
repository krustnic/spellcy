#Spellcy

Расширение для браузера Chrome, позволяющее осуществлять проверку орфографии на произвольном сайте. 
Удобно при ведении своего блога или при разработке веб-проектов. Установить расширение можно из [Chrome Web 
Store](https://chrome.google.com/webstore/detail/spellcy/dojgfgknadakhfnaogaipdbdamepeffe?utm_source=chrome-ntp-icon).

##Разработка
Расширение написано на EcmaScript 6 при помощи [Babel](https://babeljs.io/) и [Browserify](http://browserify.org/).
#####Компиляция:
```
grunt build
```
#####Прогон тестов
```
grunt test
```
#####Сборка zip-архива расширения
```
grunt production
```
